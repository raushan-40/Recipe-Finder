# 🍽️ Recipe Finder

A modern and responsive **Recipe Finder** web application built using **HTML, CSS, and Vanilla JavaScript**. The application allows users to search for recipes by name, fetches real-time data from a public recipe API, and displays detailed recipe information in a clean and user-friendly interface.

# ⚙️ How It Works

1. Enter a recipe name.
2. Click the **Search** button.
3. The application sends a request to the Recipe API.
4. Matching recipes are displayed.
5. Click a recipe to view detailed information.

---

# 📡 API Used

This project uses a public Recipe API to fetch recipe data.

Example Endpoint:

```
https://www.themealdb.com/api/json/v1/1/search.php?s=Arrabiata
```

The API returns:

- Recipe Name
- Image
- Ingredients
- Instructions
- Category
- Area
- YouTube Link (if available)
